# NebulaOps v23.1 — Fix 1-13 Implementation Report

## Scope

This package applies the 13 requested fixes while keeping the current extension strategy: APIForge, KubeBridge and Contract Hub are controlled from the Extensions UI, and no extension data is mocked or seeded.

## Implemented fixes

1. **AI Engine Anthropic integration** — `ai-engine/app/main.py` now calls Anthropic Messages API when `ANTHROPIC_API_KEY` is configured. If the provider is unavailable, the service returns an explicit `LLM_UNAVAILABLE` fallback based only on caller-provided runtime evidence.
2. **Gateway HealthController v23.1** — `/api/health` and `/api/ping` now report `23.1` and `NebulaOps v23.1`.
3. **AiOpsController cleanup** — `RestTemplate` is injected, Authorization/correlation headers are forwarded, and AI/local analysis is consolidated through a single `analyzeThroughEngine` path with deterministic fallback.
4. **Test suites** — Added test suites for `ai-ops-service`, `auth-service` and `gateway-service`.
5. **ai-ops-service test dependencies** — Added `spring-boot-starter-test` and `mockito-junit-jupiter`.
6. **Shared Kernel** — Added `AbstractExtensionController`, `ExtensionCapability`, `ExtensionRuntimeStatus` and `NebulaHttpClient`.
7. **Go cache observability** — Added `ObservabilityCache` to the Go cache service and gateway-side `ObservabilityCacheClient` for Loki/Prometheus query caching.
8. **APIForge PVC data dir** — `APIFORGE_DATA_DIR` now points to the PVC-backed mount `/var/lib/apiforge` in Docker and Kubernetes.
9. **GitHub CI** — Backend tests are enabled and a Trivy filesystem security scan is added.
10. **GitLab CI** — Added `security` stage and `security:trivy-fs` job.
11. **.env.example** — Documented `ANTHROPIC_API_KEY`, `ANTHROPIC_MODEL` and timeout settings.
12. **application.yml warnings** — Added explicit warnings for development JWT secret and `KEYCLOAK_AUTH_ENABLED=false`.
13. **Frontend typed API client** — `frontend/src/app/api.config.ts` now exports `NebulaApiClient` with typed methods for platform, runtime, AI Ops, extensions and service APIs.
