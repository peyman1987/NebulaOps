# NebulaOps UI Kit

Shared frontend primitives for NebulaOps v23.3 micro frontends.

This package is intentionally lightweight and framework-aligned with the existing Angular 18 remotes. It defines reusable UI contracts and CSS tokens for:

- status pills
- metric cards
- timeline entries
- entity tables
- confirmation modal patterns
- code/YAML viewer contracts
- command palette result contracts

The shell and remotes can gradually consume this library to avoid duplicating visual and interaction primitives.
