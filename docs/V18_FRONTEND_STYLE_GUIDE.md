# Frontend v18 Style Guide

Il frontend v18 usa un look dark, holographic e operativo. L'obiettivo non è solo estetico: ogni tab mostra una funzione
concreta.

## Tab principali

- **OVERVIEW**: hero animato, metriche, service galaxy, SVG architetturale.
- **TASKS**: Kanban CRUD con drag & drop.
- **KUBERNETES**: CRUD risorse, filtri, YAML live editor, scale.
- **TERRAFORM**: moduli IaC, drift, risorse, plan preview.
- **HELM**: release lifecycle.
- **OBSERVABILITY**: log stream con auto-refresh.
- **CICD**: pipeline GitLab/ArgoCD.
- **SECURITY**: checklist DevSecOps.
- **FINOPS**: costi e regole di ottimizzazione.
- **BACKUPS**: playbook backup/restore.
- **DOCS**: indice documentazione.

## Linee guida visuali

- Sfondo dark con gradienti profondi.
- Card glassmorphism.
- Badge luminosi.
- Diagrammi SVG animati.
- Orbit e flow animati nella hero.
