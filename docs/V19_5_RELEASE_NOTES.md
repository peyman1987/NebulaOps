# NebulaOps v19.5 Release Notes

Advanced Observability Stack, GitOps Control Plane, Multi-Environment Manager and Smart Terraform Studio. INFRA links
now include Grafana, Redis, MongoDB, RabbitMQ, Prometheus, Loki, Tempo, OpenTelemetry and ArgoCD.

## Portfolio value

This v19.5 module is designed as an enterprise DevOps/SRE portfolio feature with animated frontend UX and backend
simulation APIs.

## v19.5 refresh — Docker Desktop + OpenLens integration

- Added `CONTAINERS` tab.
- Added Docker Desktop-like container runtime console.
- Added local container actions: start, stop, restart, pause and prune safe cache.
- Added image, volume and network visibility.
- Added OpenLens-like Kubernetes workload console.
- Added pod scaling, rollout restart, ingress/service inspect and controller actions.
- Added integrated Docker + kubectl terminal preview.
- Extended INFRA hub with Docker Desktop Console and OpenLens Console links.
