# NebulaOps v20.2 Diagrams

Diagrammi aggiornati per la versione **v20.2 Full Platform**.

Questa release visualizza la nuova architettura con dashboard principale, sidebar laterale, pagine dedicate, gateway,
servizi Java, worker Go, AI engine Python e integrazioni runtime Docker/Kubernetes/GitOps/Observability.

## Diagrammi inclusi

- Full Platform Overview
- AI Ops Architecture
- Kubernetes Visual Cluster
- DevSecOps Module
- CI/CD Pipeline Designer
- INFRA Hub
- Observability Stack
- GitOps Control Plane
- Multi-Environment Manager
- Smart Terraform Studio

Tutti gli SVG sono disponibili sia in `docs/diagrams/` sia in `frontend/src/assets/`, così la home Angular e la
documentazione usano gli stessi artefatti aggiornati.
