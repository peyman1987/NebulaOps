# NebulaOps documentation

This directory contains the aligned documentation for the NebulaOps v23.4 same-origin local runtime.

| File | Purpose |
| --- | --- |
| `TECHNICAL_DOCUMENTATION.md` | Runtime topology, reverse proxy, services, identity and delivery model |
| `API_EXAMPLES.md` | Gateway/API request examples |
| `DEPLOYMENT_GUIDE.md` | Docker Compose, WSL, Helm and optional GitLab/Argo CD guidance |
| `GITLAB_ARGOCD.md` | CI/CD and GitOps workflow |
| `TROUBLESHOOTING.md` | Local startup and health diagnostics |
| `WSL_GUIDE.md` | WSL-oriented execution notes |
| `diagrams/` | Canonical SVG diagrams validated by the preflight |

The supported browser entry point is `http://nebulaops.localhost`.
