# NebulaOps v22.5 — APIForge-only extension package

This package keeps APIForge as the only Kubernetes-hosted extension. The other extension projects, Kubernetes manifests and APP BAR entries have been removed.

## Kept

- `extensions/apiforge`
- `extensions/apiforge/k8s/deployment.yml`
- APIForge APP BAR card on port `31110`
- `./scripts/wsl/start.sh --rebuild` auto-deploy path for APIForge

## Removed

- KubeBridge
- Runbook Center
- Extension Registry
- Contract Hub
- EventOps Center
- Observability Lens
- GitOps Center
- Secrets & Config Center
- SLO Center
- Backup & Recovery Center

## Runtime policy

APIForge remains a Spring Boot MVC extension deployed to Kubernetes. No additional extension projects are bundled.
